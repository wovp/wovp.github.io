这是我的第一篇博客

记录一个下午遇到的问题

起因：搭建博客

第一个ques：在配置node全局缓存时将根目录下的modules文件转移到了自建的global文件里，最终导致环境变量混乱，不得已卸载node重装

注意点：建立global的时候，在里面再创建一个modules空文件夹

第二个ques：不知道如何创建文本上传

注意点：git bash里输入 hexo new +文件名

第三个ques：在最后上传文件的时候碰到了github连接仓库失败

解：重新连一下就好了。用git bash在根目录：git config –global http.sslVerify “false”

扩展：因觉得每次修改后都要hexo clean—hexo g—hexo d比较繁琐，索性自己在桌面上创建了一个脚本bat：@echo off E: cd E:\Blog\blog5 hexo clean&&hexo g&&hexo d
